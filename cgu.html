1 Présentation de l'application

Fläag est une application mobile de messagerie géolocalisée. Elle permet aux utilisateurs de déposer des messages texte, photo ou audio ancrés à des coordonnées GPS précises. Ces messages ne peuvent être consultés qu'en se trouvant physiquement à proximité du point de dépôt (dans un rayon de 100 mètres).

L'application est éditée et exploitée par Guillaume Garth, ci-après dénommé « l'Éditeur ». Pour toute question : support@flaag.app.

Les présentes Conditions Générales d'Utilisation (CGU) régissent l'accès et l'utilisation de l'application Fläag, disponible sur iOS (App Store) et Android (Google Play).

3 Création de compte

L'accès aux fonctionnalités de Fläag nécessite la création d'un compte. L'inscription est possible via :

  • Numéro de téléphone avec code OTP (mot de passe à usage unique)
  • Compte Google (OAuth)

Vous êtes responsable de la confidentialité de vos identifiants. Vous vous engagez à nous signaler immédiatement tout accès non autorisé à votre compte.

Un seul compte est autorisé par utilisateur. La création de comptes multiples ou de faux comptes est interdite.

6 Géolocalisation

Fläag repose sur la géolocalisation pour fonctionner. L'application peut accéder à votre position :

  • Au premier plan : lorsque l'application est ouverte, pour afficher les messages proches sur la carte
  • En arrière-plan : si vous l'autorisez explicitement, pour vous notifier lorsque vous passez à proximité d'un message
La géolocalisation en arrière-plan est optionnelle. Vous pouvez désactiver cette permission à tout moment dans les réglages de votre appareil. Cela désactivera uniquement les notifications de proximité — le reste de l'application fonctionnera normalement.

Votre position n'est jamais partagée publiquement avec d'autres utilisateurs. Elle est utilisée uniquement pour déterminer votre proximité avec des messages.

7 Politique de confidentialité

La présente politique de confidentialité décrit la manière dont Fläag collecte, utilise et protège vos données personnelles, conformément au Règlement Général sur la Protection des Données (RGPD - Règlement UE 2016/679).

Base légale du traitement

Le traitement de vos données repose sur :

  • L'exécution du contrat (CGU) que vous avez accepté en vous inscrivant
  • Votre consentement explicite (notamment pour la géolocalisation en arrière-plan)
  • Notre intérêt légitime (sécurité, prévention des abus)

8 Vos données personnelles

Données collectées

  • Données d'identification : numéro de téléphone ou email Google, nom d'affichage, photo de profil
  • Données de localisation : coordonnées GPS au moment du dépôt d'un message, position lors de la lecture
  • Contenu : messages texte, photos, enregistrements audio que vous publiez
  • Données d'usage : interactions avec l'application, horodatages, logs d'erreurs techniques
  • Token push : identifiant technique pour l'envoi de notifications

Utilisation des données

  • Affichage des messages géolocalisés sur la carte
  • Envoi de notifications de proximité
  • Messagerie entre utilisateurs
  • Amélioration de l'application (données anonymisées)
  • Prévention des abus et sécurité

Conservation des données

Vos données sont conservées aussi longtemps que votre compte est actif. En cas de suppression de compte, vos données personnelles sont supprimées dans un délai de 30 jours, à l'exception des données imposées par la loi.

Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : s'opposer à certains traitements

Pour exercer ces droits : support@flaag.app

Partage des données

Vos données ne sont jamais vendues à des tiers. Elles peuvent être partagées avec :

  • Supabase (hébergement base de données, UE) — politique de confidentialité
  • Expo / EAS (service de notifications push)
  • Google (cartographie, authentification OAuth) — uniquement si vous utilisez la connexion Google

Sécurité

Toutes les communications entre l'application et nos serveurs sont chiffrées (HTTPS/TLS). Les données sensibles (tokens d'authentification) sont stockées dans le trousseau sécurisé de votre appareil. Une politique de contrôle d'accès (Row Level Security) est appliquée sur toutes les tables de la base de données.
